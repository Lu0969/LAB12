Лабораторная работа №12 
Работа с лямбда-выражениями. 

В данном проекте представлено решение следующих задач:
1. создание простого лямбда-выражения
2. создание лямбда-выражения с одним параметром
3. создание лямбда-выражения с двумя параметрами
4. создание блока лямбда-выражений
